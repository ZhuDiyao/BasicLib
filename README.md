# BasicLib
This project is led by ZhuDiyao. The founders are ZhuDiyao and ChenJiancheng. It was originally applied to the project of ZheJiangJiaXuanWuLiu.

# 此项目本应用于浙江佳轩物流有限公司的仓储管理项目。

# 解决方案包含了 五个模块

# 第一个 JX_StandardLibrary_WMS
此项目主要定义了一些标准值、标准提示方法，应用于项目

# 第二个 JX_StandardMethod_WMS
此项目主要定义了一些标准方法，可以方便的实现某些功能

# 第三个 JX_TCPIPDataProtocol
此项目主要定义了 ICP/IP传输协议的方法，鼠标键盘模拟操作和记录的相关功能

# 第四个 测试启动项
用于启动项目，并测试

# 第五个 静态类库测试
无用的空项目
